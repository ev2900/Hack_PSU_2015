What We Do

Our Projects

Sexual Assuly is a problem at universities around the United States. Penn State is no exception. You, me a freind, a family member could be the victim of sexual assualt. We all get the texts. We all get the emails but what are we doing about this issue? Change start with each and everyone of us. You and me. Awarness starts with an understanding of the problem. Our project compiles the data on sexual assults to provide an accureate, unbiased and sounds representation of our problem. The university publishes information everytime an assualt happens however the information is in an unfreindly and unintitatuve format. We have complied the data from every Penn State campus and we have provided an intuative map with the same information as each notice from University Services. The mobile freindly app and scalable app give Penn State the ability to frame our problem on a more effective platform.

We want to make a difference. Preventing sexual assult is the task. Awarness is a start. The tool provides a place to start.

What Technology





Data + Tech = Change



Technolgoy can provide a solution. Our platform is a web application. It utalizes the mapbox API and Amazon Web Services to bring this website here. Using the mapbox API the map of the assult is easily updateable. Using Amazon Web Service the tool is easily scalable and redeployable.

Open source and on a substainable platform any university for free can set up this same tool in only moments. Hosted on AWS , visualized with MapBox and open sourced on GitHub.
